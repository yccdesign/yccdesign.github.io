About this site.
This portfolio website has been an ongoing side project since 2018.

The site is written in HTML, CSS, and some JavaScript, with a lot of help from W3Schools and Stack Overflow. My tools include Atom, Github, and Illusrtator.

The website has gone through multiple revamps, for which I have added some cool JS effects such as the scroll progress bar, as well as more visual elements, to make it more user-friendly while representing my work and personality.

Please feel free to email me if you spot any bugs or have any suggestions! And of course, this website will keep evolving.
