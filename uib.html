BACKGROUND

What is UI Builder?


UI Builder allows users to build pages for ServiceNow experiences. It is a WYSIWYG, low-code tool to reduce time needed to create interactive web apps on the Now Platform.

UI Builder allows users to build a ServiceNow app with 115+ customizable components in the Rome release.
THE CHALLENGES

It's more than a design project


Based on previous research insights, the team identified two major topics to tackle when it comes to configuring components:

  1. The input mode switcher needed to be enhanced for usability and scalability
  2. Users are confused with arbitrary terms and lack of helper experience when configuring components
  3. Inconsistencies across different components' configuration needed to be resolved


Insights gathered from previous usability research served as the foundation for an enhancement project for the configuration panel
